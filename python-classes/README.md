# Python Classes and Objects

This project contains implementations of a Square class with progressive levels of complexity.

## Files

- `0-square.py` - Empty Square class
- `1-square.py` - Square with private size attribute
- `2-square.py` - Square with size validation
- `3-square.py` - Square with area calculation
- `4-square.py` - Square with property getter and setter
- `5-square.py` - Square with printing capability
- `6-square.py` - Square with position support

## Learning Objectives

- Understand OOP concepts
- Work with classes and objects
- Implement private attributes and properties
- Use getters and setters
- Understand data encapsulation
- Master the `__init__` method
