# JavaScript - Warm Up

This repository contains JavaScript exercises for beginners covering fundamental concepts.

## Requirements

- Allowed editors: vi, vim, emacs
- All files will be interpreted on Ubuntu 20.04 LTS using node (version 14.x)
- All files should end with a new line
- The first line of all files should be exactly `#!/usr/bin/node`
- Your code should be semistandard compliant (version 16.x.x)
- All files must be executable
- No variable declarations allowed (const/let only, no var)

## Installation

To run the scripts, you need Node.js 14.x installed:

```bash
curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -
sudo apt-get install -y nodejs
```

To check code style with semistandard:

```bash
sudo npm install semistandard --global
```

## Files

- `0-javascript_is_amazing.js` - Print a constant string
- `1-multi_languages.js` - Print multiple languages
- `2-arguments.js` - Arguments handling
- `3-value_argument.js` - Print first argument
- `4-concat.js` - Print two arguments in a formatted string
- `5-to_integer.js` - Convert argument to integer
- `6-multi_languages_loop.js` - Loop to print languages
- `7-multi_c.js` - Print "C is fun" multiple times
- `8-square.js` - Print a square with X characters
- `9-add.js` - Addition of two integers
- `10-factorial.js` - Recursive factorial calculation
- `11-second_biggest.js` - Find second biggest integer
- `12-object.js` - Modify object property
- `13-add.js` - Module exporting add function

## Usage

To run a script:

```bash
./0-javascript_is_amazing.js
./2-arguments.js Best School
```

To check code style:

```bash
semistandard *.js
```
