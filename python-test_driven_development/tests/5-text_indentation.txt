The ``text_indentation`` module
=================================

Using ``text_indentation``
--------------------------

Import the function:

    >>> text_indentation = __import__('5-text_indentation').text_indentation

Test basic text with period:

    >>> text_indentation("Hello.")
    Hello.
    <BLANKLINE>

Test with multiple sentences:

    >>> text_indentation("Hello. How are you? I am fine.")
    Hello.
    <BLANKLINE>
    How are you?
    <BLANKLINE>
    I am fine.
    <BLANKLINE>

Test with all special characters:

    >>> text_indentation("Question? Answer: Done.")
    Question?
    <BLANKLINE>
    Answer:
    <BLANKLINE>
    Done.
    <BLANKLINE>

Test with spaces after special characters:

    >>> text_indentation("Hello.    How are you?")
    Hello.
    <BLANKLINE>
    How are you?
    <BLANKLINE>

Test with no special characters:

    >>> text_indentation("Hello World")
    Hello World

Test with empty string:

    >>> text_indentation("")

Test with only spaces:

    >>> text_indentation("   ")
       

Test text is not a string:

    >>> text_indentation(123)

Test with newlines in text:

    >>> text_indentation("Hello.\nWorld")
    Hello.
    <BLANKLINE>
    World

Test with tab character:

    >>> text_indentation("Hello.\tWorld")
    Hello.
    <BLANKLINE>
    World
