# Python - Serialization

This project covers serialization and deserialization in Python using different formats.

## Files

- `task_00_basic_serialization.py` - Basic JSON serialization and deserialization
- `task_01_pickle.py` - Pickle serialization for custom objects
- `task_02_csv.py` - Convert CSV to JSON format
- `task_03_xml.py` - XML serialization and deserialization
