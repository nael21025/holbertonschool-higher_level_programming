# Python - More Data Structures: Set, Dictionary

This directory contains solutions for Holberton School's Python more data structures exercises. The lessons focus on:

- **Lists**: Methods like append, extend, insert, remove, pop, clear, index, count, sort, reverse, and copy
- **List Comprehensions**: Creating concise lists with filtering and transformations
- **Stacks and Queues**: Using lists as LIFO/FIFO data structures
- **Tuples**: Immutable sequences, tuple packing and unpacking
- **Sets**: Unordered collections with mathematical operations (union, intersection, difference)
- **Dictionaries**: Key-value pairs with methods like keys, values, items, get, update, pop
- **Lambda Functions**: Anonymous functions used with map, filter, reduce
- **Looping Techniques**: enumerate, zip, reversed, sorted for iteration

## Exercises

### 0. Squared simple
**File**: `0-square_matrix_simple.py`
- Compute the square value of all integers in a 2D matrix
- Returns new matrix without modifying the original

### 1. Search and replace
**File**: `1-search_replace.py`
- Replace all occurrences of an element by another in a new list

### 2. Unique addition
**File**: `2-uniq_add.py`
- Add all unique integers in a list (only once for each integer)

### 3. Present in both
**File**: `3-common_elements.py`
- Return a set of common elements in two sets (intersection)

### 4. Only differents
**File**: `4-only_diff_elements.py`
- Return a set of all elements present in only one set (symmetric difference)

### 5. Number of keys
**File**: `5-number_keys.py`
- Return the number of keys in a dictionary

### 6. Print sorted dictionary
**File**: `6-print_sorted_dictionary.py`
- Print a dictionary by ordered keys (alphabetic order)

### 7. Update dictionary
**File**: `7-update_dictionary.py`
- Replace or add key/value pairs in a dictionary

### 8. Simple delete by key
**File**: `8-simple_delete.py`
- Delete a key in a dictionary (with safety check)

### 9. Multiply by 2
**File**: `9-multiply_by_2.py`
- Return a new dictionary with all values multiplied by 2

### 10. Best score
**File**: `10-best_score.py`
- Return the key with the biggest integer value (highest score)

### 11. Multiply by using map
**File**: `11-multiply_list_map.py`
- Return a list with all values multiplied by a number using map (max 3 lines)

### 12. Roman to Integer
**File**: `12-roman_to_int.py`
- Convert a Roman numeral to an integer (1 to 3999)
- Handles subtractive notation (IV, IX, XL, etc.)

## Requirements

- **Language**: Python 3.8.5+
- **Editor**: vi, vim, or emacs
- **Style Guide**: pycodestyle 2.7.*
- **Shebang**: `#!/usr/bin/python3` (first line of all files)
- **Executable**: All files must be executable
- **New Line**: All files must end with a newline

## Usage

Each file can be tested with its corresponding main file:

```bash
python3 0-main.py
python3 1-main.py
# etc...
```
