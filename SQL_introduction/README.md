# SQL Introduction

This project covers the basics of SQL and databases.

## Learning Objectives

- Understand what databases and relational databases are
- Learn SQL fundamentals (CREATE, SELECT, INSERT, UPDATE, DELETE)
- Master database and table management
- Use WHERE, ORDER BY, GROUP BY, and aggregate functions

## Files

- `0-list_databases.sql` - Lists all databases
- `1-create_database_if_missing.sql` - Creates a database
- `2-remove_database.sql` - Deletes a database
- `3-list_tables.sql` - Lists tables in a database
- `4-first_table.sql` - Creates first_table
- `5-full_table.sql` - Shows table structure
- `6-list_values.sql` - Lists all rows in table
- `7-insert_value.sql` - Inserts a row
- `8-count_89.sql` - Counts records by id
- `9-full_creation.sql` - Creates second_table with data
- `10-top_score.sql` - Lists records sorted by score
- `11-best_score.sql` - Lists records with score >= 10
- `12-no_cheating.sql` - Updates a record
- `13-change_class.sql` - Deletes low scores
- `14-average.sql` - Calculates average score
- `15-groups.sql` - Groups and counts by score
- `16-no_link.sql` - Lists records with non-null names

## Usage

```bash
cat 0-list_databases.sql | mysql -uroot -p
```
