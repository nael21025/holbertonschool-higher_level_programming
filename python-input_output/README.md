# Python - Input/Output

This project covers file operations and JSON serialization in Python.

## Files

- `0-read_file.py` - Read a text file and print to stdout
- `1-write_file.py` - Write a string to a text file
- `2-append_write.py` - Append a string to a text file
- `3-to_json_string.py` - Convert object to JSON string
- `4-from_json_string.py` - Convert JSON string to object
- `5-save_to_json_file.py` - Save object to JSON file
- `6-load_from_json_file.py` - Load object from JSON file
- `7-add_item.py` - Add arguments to a list and save to file
- `8-class_to_json.py` - Convert class instance to dictionary
- `9-student.py` - Student class with JSON serialization
- `10-student.py` - Student class with filtered JSON serialization
- `11-student.py` - Student class with JSON reload
- `12-pascal_triangle.py` - Pascal's triangle implementation
